# 编辑修改与自我修订

## 自我编辑流程：大修 → 中修 → 小修

### 第一轮：大修（结构层面）
- **检查整体结构**：三幕是否完整？节奏是否合理？
- **检查三线并行**：主线、支线、暗线是否都推进了？
- **检查情节逻辑**：是否有前后矛盾或漏洞？
- **检查人物弧光**：主角和重要配角是否有成长？

### 第二轮：中修（章节层面）
- **检查章节钩子**：每章结尾是否有悬念或转折？
- **检查章节节奏**：是否有过于平淡的连续章节？
- **检查爽点分布**：爽点间隔是否合理？太长会冷场，太短会疲劳
- **检查代入感**：是否过度使用了全知视角？

### 第三轮：小修（文字层面）
- **精简冗余**：删掉可有可无的句子
- **优化句式**：长短句交替是否自然？
- **检查对话标签**：是否有太多"某某说""某某道"？
- **检查形容词**：是否可以用具体的细节代替？

## ★ 三要素自检清单

### 爽点自检
- [ ] 这一章最"爽"的地方在哪里？
- [ ] 爽点到来的节奏是否恰当？
- [ ] 压抑的深度是否足够支撑反弹的强度？
- [ ] 爽点是否太过重复（同一种爽点用多了会疲劳）？

### 代入感自检
- [ ] 读者在这一章是否一直在主角视角里？
- [ ] 主角的反应是否合理？像一个普通人会做出的反应吗？
- [ ] 是否有破坏代入感的"作者旁白"？
- [ ] 读者会不会觉得"如果是我，我也会这么做"？

### 期待感自检
- [ ] 这一章结束时读者是否想知道"后来呢"？
- [ ] 是否有新的悬念或目标埋下？
- [ ] 长期目标和短期目标是否都在推进？
- [ ] 危机感是否在持续？

## ★ 字数优化原则

### 先删这些（废话）
- 重复说明（同一个设定解释了两次）
- 无意义的过渡（"他吃完饭，站起来，走向门口"→"他走向门口"）
- 多余的修饰语（"他非常极其特别愤怒"→"他怒了"）
- 全面的环境描写（选一个细节就够了）

### 保留并强化这些（血肉）
- 心理描写的深度
- 人物互动的完整性和生动性
- 爽点/代入感/期待感相关段落

### 核心口诀
> **删"废话"而非"血肉"——不求字数少，求每个字都有存在的理由。**

## 常见问题检查清单

### 对话问题
- [ ] 对话标签是否单调重复？
- [ ] 对话是否太过直接（没有潜台词）？
- [ ] 每个角色的说话风格是否可区分？

### 描写问题
- [ ] 是否有大段的"无用描写"？
- [ ] 描写是否服务于情绪？
- [ ] 是否过度依赖形容词？

### 节奏问题
- [ ] 是否有连续多章没有推进任何线索？
- [ ] 紧张场景是否用了短句？
- [ ] 情感段落是否给了足够的空间？

### 逻辑问题
- [ ] 时间线是否有矛盾？
- [ ] 设定前后是否一致？
- [ ] 人物行为是否符合性格？

## Beta 阅读反馈处理

1. **收集反馈时**：不辩解、不解释，认真倾听
2. **分类处理**：
   - 多人提到的同一个问题 → 必须改
   - 单人提到的感觉问题 → 考虑是否要改
   - 不符合你想法的建议 → 保留自己的判断
3. **改完后**：不需要对所有反馈说"已修改"，好的作品源于好的判断

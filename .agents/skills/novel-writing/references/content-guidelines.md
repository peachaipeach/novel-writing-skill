# 内容安全规范

## 都市题材命名规范

- **不使用现实中真实存在的城市名、地名**
- 虚构城市名应避免与现实地名过于相似
- 使用虚构的路名、区域名
- 如需使用现实地理框架（如"北方某城市"），确保具体地点为虚构

### 示例
| ❌ 避免 | ✅ 使用 |
|---------|---------|
| 上海市 | 海城 / 临州市 |
| 北京朝阳区 | 北城朝阳区（完全虚构的命名） |
| 南京路 | 江滨路（通用名称） |

## 历史与现实人物处理

- **正面历史人物不得丑化**，尊重历史评价
- 虚构人物避免与历史人物过度关联
- 涉及真实历史事件时，不要篡改关键史实
- 当代现实人物（名人、公众人物）不要作为角色出现

## 社会黑暗面把控

- **可适度提及，但不可过度渲染**
- 避免大量负面情节堆砌
- 黑暗描写的目的是衬托光明，而非展示黑暗本身
- 涉及暴力、犯罪等内容时，不应有详细的过程描写

### 原则
- 点到为止，不细描
- 服务主题，不渲染
- 黑暗要有"度"，不能让人读完感到绝望

## 平等原则

- 角色设定和情节不得包含性别歧视
- 角色设定和情节不得包含种族歧视
- 角色设定和情节不得包含地域歧视
- 不得有任何形式的歧视性表述或暗示

### 注意
- 反派可以是任何性别、种族，但不要将反派身份与性别/种族挂钩
- 角色设定应多样化，避免刻板印象
- 敏感话题需要谨慎处理

## 宗教敏感

- 涉及宗教时应尊重，不可亵渎或歪曲
- 避免虚构宗教影射现实宗教
- 避免宗教争议内容
- 宗教元素应服务于故事，而非作为批判对象

### 建议
- 如需使用宗教元素，采用完全虚构的宗教体系
- 如果必须在现实宗教背景下写作，确保内容准确且尊重
- 不确定时，宁可模糊处理也不冒险

## 编辑阶段的安全自检

在编辑修改阶段加入以下内容安全自检：

- [ ] 全部地名是否为虚构？
- [ ] 是否有任何可能被视为丑化历史人物的内容？
- [ ] 社会黑暗面是否在合理范围内？
- [ ] 所有角色设定是否贯彻了平等原则？
- [ ] 宗教相关内容是否经过慎重处理？

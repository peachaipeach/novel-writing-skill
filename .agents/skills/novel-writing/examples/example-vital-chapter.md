# 示例章节：人物驱动写法 vs 结构驱动写法对比

> 本文档展示了同一个场景的两种写法对比，重点突出 **v3.0 人物驱动技法** 与原有写法的差异。
>
> 左侧为现有写法（example-chapter-annotated.md 风格），右侧为人物驱动重写版。
> 注释说明人物鲜活度和沉浸式叙事技法的具体运用。

---

## 场景相同：苏明第一次在考试中使用预知能力

---

### 写法 A（结构驱动版）— 来自 example-chapter-annotated.md

```
考试铃响的时候，苏明还在想昨天那件事。

监考老师抱着一沓卷子走进来，皮鞋敲在水磨石地面上的声音让整间教室安静下来。

"期中摸底测试，全校排名。"老师推了推眼镜，目光若有若无地扫过苏明的位置，
"这次考试成绩会直接影响分班。"

教室里响起一阵低低的议论声。苏明的前桌转过头来，脸上带着幸灾乐祸的笑：
"明哥，这次你准备考多少分啊？"
```

> **特点**：叙事清晰、信息效率高、一章推进了"考试→发现能力→提前交卷→被组织盯上"的完整剧情。
> **问题**：节奏均匀（从头到尾同一距离）、角色功能化（在"完成剧情任务"）、缺少身体存在感。

---

### 写法 B（人物驱动版）— 同一场景重写

```
铃响了。

苏明没动。他坐在靠窗第三排，手插在口袋里，指尖抵着打火机的金属壳。
冰的。他把它翻了个面。

——哒。
又翻了个面。
——哒。

"期中摸底测试，全校排名。"

老师的声音从讲台那边传过来，像是隔了一层水。
苏明抬起头，看见老师的嘴在动，但那些字一个一个落进耳朵里，没有组成意思。

……全校排名。

他把打火机翻了个面。

（他应该把它放在家里的。他告诉自己说要放下的。
    但他没有。
    他不知道自己是忘了，还是故意没忘。）

"这次考试成绩会直接影响分班。"

前桌转过头来。
苏明看到了他的嘴在咧开，露出一个他很熟悉的表情。
"明哥，这次你准备考多少分啊？"

苏明看着他。
他想说"关你什么事"。
但他没说。他低下头，继续转那个打火机。

——哒。
——哒。

他自己也不知道自己是在等什么。
等卷子发下来？
还是在等昨天那个实验里发生的事情再发生一次？

（他害怕它不会再来。
    他也害怕它再来。）

他把打火机握在手心里，握紧了。
金属壳硌着掌心，疼。
但他没有松开。
```

> **技法运用说明**：
>
> **① 身体存在感**：开头就给出了角色的触觉（指尖抵着打火机的金属壳、冰的）、听觉（铃响、声音像隔着水）、痛觉（硌着掌心，疼）。角色立即有了物理存在。
>
> **② 意图与行动的裂缝**：把打火机翻过来翻过去——他在做一件毫无意义的事，因为他脑子里在想别的事。他想说"关你什么事"——但没说。这种"想做但没做"的裂缝就是人物鲜活的来源。
>
> **③ 人物不自知的东西**：（他害怕它不会再来。他也害怕它再来。）——角色自己在心里同时承认了两个相反的恐惧。读者感受到他的矛盾，比被告知"他既期待又害怕"更真实。
>
> **④ 叙事距离拉近**：不写"苏明心不在焉"，直接写动作（翻打火机、哒、哒）和破碎的感官（声音像隔着水）。读者不是在"看"他心不在焉，而是和他一起"经历"心不在焉。
>
> **⑤ 括号心理空间**：用括号和斜体标注角色内心深处的真实想法。这是"角色说给自己听的话"——不是解释给读者听的说明文。
>
> **⑥ 句式服务于情绪**：短句（"铃响了。""冰的。""疼。"）制造碎片感——和角色碎片化的注意力匹配。长句（"他不知道自己是忘了，还是故意没忘"）放慢节奏，带出内心的犹豫。

### 对比分析

| 维度 | 写法 A（结构驱动） | 写法 B（人物驱动） |
|------|-------------------|-------------------|
| **叙事距离** | 较远——叙述发生了什么 | 极近——体验正在发生什么 |
| **身体存在** | 几乎没有（坐在教室） | 强（触觉/听觉/痛觉/打火机的重量） |
| **人物内心** | 通过行动暗示（抠笔杆=紧张） | 直接进入但不解释（括号内的矛盾声音） |
| **意图裂缝** | 无 | 有（想怼人但没说；想放下打火机但没放下） |
| **时间流速** | 均匀推进 | 不均匀——关键瞬间（哒、哒）放慢 |
| **信息效率** | 高（一章推了大量剧情） | 低（但读者更"在"故事里） |
| **读者感受** | 知道发生了什么 | 感受到了发生了什么 |

### 进一步对比：考试过程中

```
📋 写法 A（结构驱动）：
苏明深吸一口气，把注意力拉回到纸上。
第一题。不会。第二题。也不会。
……

📋 写法 B（人物驱动）：
他低头看卷子。
第一道题。
那些字他认识，但连在一起——他读了三遍才读进去。
（以前不是这样的。
    以前他扫一眼就知道答案。
    那是……多久以前了？）
他眨了一下眼。视线有点模糊。
他用手指按住太阳穴，按了两秒。
（好了。看下一道。）
第二道题。
还是不会。
他的指甲在卷子边缘来回刮，发出细细的沙沙声。
```

**关键差异**：
- 写法 B 的"读了三遍才读进去"是具体的行为 + 时间流逝的质感
- 写法 B 的"（以前不是这样的）"是角色心中一闪而过的念头——不是为了解释给读者听，而是这个人在此刻的真实心理活动
- "用指甲刮卷子边缘"是一个微行为，角色自己都没意识到自己在做这个动作

### 进一步对比：使用能力时

```
📋 写法 A（结构驱动）：
只要他集中注意力盯着卷子，那些原本模糊的文字就会像被水洗过一样变得清晰，
然后答案就从脑子里"长"出来。

📋 写法 B（人物驱动）：
他盯着卷子。
没用。
他闭了一下眼。
（行不行？）
再睁开。
这一次不一样了。
那些字没有变清晰——不是清晰——是——
它们自己"动"了。
不是真的在动。他脑子里有另一个声音，在读那些题目，一字一句，
然后——
答案。
不是他想出来的。是"出现"的。像有人在他耳边说了一句悄悄话。
他的手开始发麻，从指尖往上，到手腕，到小臂。
不是麻。
是一种……凉。
```
```

**关键差异**：
- 写法 A 直接告诉读者能力是怎么运作的（作者在解释设定）
- 写法 B 让读者和角色一起"发现"能力的运作方式（角色在经历未知）
- 写法 B 加入了身体感（手发麻、凉）——能力不仅有功能效果，也有身体体验

### ★ 互动密度对比（v3.1 新增）

上述示例中，角色之间的直接互动很少（主角大部分时间在和自己互动）。以下是**专门的互动场景对比**，展示"一笔带过"和"连续展开"的差异。

#### 场景：前桌挑衅苏明

```
📋 薄互动（1回合收——典型的一笔带过）：
前桌转过头来，脸上带着幸灾乐祸的笑："明哥，这次你准备考多少分啊？"
苏明没理他。

📋 厚互动（4回合展开——连续来回）：
前桌转过头来，脸上带着幸灾乐祸的笑。
（轮1开始）
"明哥，这次你准备考多少分啊？"
苏明看了他一眼。没说话。

"我猜30分？不对，20分？"前桌笑得更大声了。
（轮2：前桌加码——嘲笑升级）

苏明把打火机翻了个面。哒。
"你好像很关心我。"
（轮3：苏明回应——不是忍让也不是爆发，是用冷处理反击）

前桌的笑容僵了一下。"谁关心你了。"
他转回去了。
（轮4：前桌退缩——互动结束，双方位置交换了）
```

**差别**：
- 薄互动：挑衅→无视。读者知道"前桌嘲笑他，他没理"。
- 厚互动：挑衅→沉默→嘲笑升级→冷回应→对方退缩。**读者看到了一个完整的互动过程**，而不仅仅是结果。

注意：厚互动并没有增加情节推进，只是让同一个情节的互动从"报告结果"变成了"展开过程"。

#### 场景密度分析表

| 版本 | 来回数 | 每回合厚度 | 读者感受到的 |
|------|--------|-----------|-------------|
| 薄互动 | 1回合 | 只有台词 | 苏明被嘲笑了 |
| 厚互动 | 4回合 | 台词+行动+潜台词 | 苏明被嘲笑→他选择冷处理→他的沉默比反击更有力→对方反而慌了 |

**关键认知**：厚互动不是多写剧情，而是**把同一个剧情"停留"更久**。

---

### 结语

> 两种写法的区别不在"谁写得好"，而在**出发点的不同**。
>
> **写法 A 问**："这章需要发生什么？怎么高效地让读者知道？"
> **写法 B 问**："这个人此刻正在经历什么？"
>
> 在日常写作中，两种写法各有用处——快节奏推进时用 A，关键情绪场景用 B。
> 但如果整本书只用写法 A，结果就是"人物僵硬、像在陈述"。
>
> **关于互动密度（v3.1 重点）**：
> - **写法 A 容易"一笔带过"**：因为它追求效率，一次交换就收
> - **写法 B 自然"互动连续"**：因为它追求体验，会在一个场景里"停留"
> - **建议**：关键互动场景至少写3个来回。即使剧情相同，读者的体验完全不同
> - 更多技法见 `references/scene-craft.md`
